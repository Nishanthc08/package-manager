# Roadmap

## v1.0 (current)
- Package builder with form validation and control file preview
- File manager with drag-drop and permission editing
- Build pipeline with progress, logs, and history
- Package manager with installed package browsing and removal
- Repository manager with GPG signing
- Template system with import/export

## Planned

### v1.1
- Dependency graph visualization
- Build queue with parallel builds
- Version comparison tool
- Script editor with bash linting

### v1.2
- CI/CD pipeline integration
- Multi-architecture builds
- Snapshot/rollback for repositories
- Plugin system for custom validators

### v1.3
- Flatpak and Snap package export
- Remote repository sync
- Collaborative package editing
